#### Installation

In your Angular project:

```
$ npm i @jchpro/cms-core
```

In `angular.json` at path `projects.<YOUR-PROJECT>.architect.build.options` add:

```json
{
  "stylePreprocessorOptions": {
    "includePaths": [
      "node_modules/@jchpro/cms-core/lib/styles"
    ]
  }
}
```
